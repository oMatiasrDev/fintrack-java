# 💰 FinTrack - Sistema de Controle Financeiro

Sistema desenvolvido em Java com o objetivo de auxiliar no controle de finanças pessoais, permitindo o gerenciamento de receitas e despesas diretamente pelo terminal.

---

## 🚀 Funcionalidades

- 📥 Cadastro de receitas  
- 📤 Cadastro de despesas  
- 📊 Geração de relatórios financeiros  
- 📅 Organização das transações  
- 💻 Interface via terminal (console)  

---

## 🛠️ Tecnologias utilizadas

- Java  
- Programação Orientada a Objetos (POO)  
- Maven
